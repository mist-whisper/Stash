# 美国
1password.com // 各子域为 1Password 提供核心服务
1passwordservices.com // 各子域为 1Password 提供支持服务，例如 Duo 集成、遥测和订阅管理。
1passwordusercontent.com // 提供 1Password 帐户的用户内容，例如保存的文件和个人资料图片。

# 其他域名
accounts.brex.com // 提供对 Brex 虚拟卡的支持
api.privacy.com // 为虚拟卡片提供支持
api.pwnedpasswords.com // 使用 haveibeenpwned.com 提供的 Pwned Passwords 服务在 Watchtower 中提供易受攻击的密码功能。
cache.agilebits.com // 提供登录名和软件许可证项目的图标
in.appcenter.ms // 在适用于 iOS 的 1Password 7 和适用于 Mac 的 1Password 7.9.10 及更早版本中提供崩溃报告管理。
jmap.fastmail.com // 为 Fastmail 中的蒙面电子提供支持
platform.brex.com // 提供对 Brex 虚拟卡的支持
1infra.net // 在 1Password 网络内提供基础设施识别
fastmail.com // 为 Fastmail 中的蒙面电子提供支持
globalsign.com // 当 1Password 应用程序请求时，提供对 Windows 上代码签名到期日期的检查。
